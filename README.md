# Experimental Investigation on Reclaimed Asphalt Pavement (RAP)

## Overview

This project investigates the performance of reclaimed asphalt pavement (RAP) in warm mix asphalt and explores sustainable approaches to road construction.


## Objectives

* Evaluate performance of RAP in bituminous mixes
* Analyze durability under different conditions
* Improve sustainability in asphalt production


## Key Concepts

* Warm mix asphalt technology
* Moisture susceptibility testing
* Sustainable material usage


## Key Work

* Conducted experiments using 100% RAP materials
* Evaluated moisture susceptibility and durability
* Explored increasing RAP usage beyond standard limits (~30%)
* Analyzed performance and structural integrity


## Results

* Demonstrated feasibility of higher RAP usage
* Improved sustainability through reduced temperature requirements
* Optimized material utilization in asphalt production
